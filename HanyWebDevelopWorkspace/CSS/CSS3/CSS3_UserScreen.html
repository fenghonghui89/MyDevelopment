box-sizing

回到目录 
   * box-sizing
    以确切的方式定义适应某个区域的具体内容
    如果不指定,则默认一行内尽量输出
    如果指定且配合width/float,则按照样式排列

    默认情况下，元素的宽度与高度计算方式如下：
    width(宽) + padding(内边距) + border(边框) = 元素实际宽度
    height(高) + padding(内边距) + border(边框) = 元素实际高度
    设置box-sizing后,则 padding(内边距) 和 border(边框) 也包含在 width 和 height 中
