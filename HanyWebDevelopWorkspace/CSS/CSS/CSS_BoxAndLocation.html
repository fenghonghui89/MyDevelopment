框模型:边框 背景 外边距 内边距

回到目录 
    * 笔记:

    背景颜色只包含内边距,不包含边框,边框如果是虚线则边框背景色也是跟元素背景色一样

    内边距、边框和外边距都是可选的，默认值是零
    外边距默认是透明的，因此不会遮挡其后的任何元素.
    外边距可以是负值，而且在很多情况下都要使用负值的外边距

    内边距padding 外边距margin 上右下左
    边框 上右下左 颜色同理 默认为none

    外边距合并(具体看w3cschool)
    指的是，当两个垂直外边距相遇时，它们将形成一个外边距。
    合并后的外边距的高度等于两个发生合并的外边距的高度中的较大者。

    例子:
    左右外边距50 虚线边框2 内边距10



相对定位 relative

回到目录 
    * 相对定位 relative
    相对于自身原始位置 所占原来空间仍然存在 后续元素不能占据



绝对定位 absolute

回到目录 
    * 绝对定位 absolute
    绝对定位的元素的位置相对于最近的已定位祖先元素，如果元素没有已定位的祖先元素，那么它的位置相对于最初的包含块
    会从正常流中删除 并且能够交叠元素 不占空间 后续元素可以占据

    相对于浏览器窗口进行定位 受四边距离限制





zindex 堆叠顺序

回到目录 
    * zindex 堆叠顺序
    Z-index 仅能在定位元素上有效 即必须设置position
    默认为auto 堆叠顺序与父元素相等
    该属性设置一个定位元素沿 z 轴的位置，z 轴定义为垂直延伸到显示区的轴。
    如果为正数，则离用户更近，为负数则表示离用户更远
