js window

回到目录 
    例如:
    显示器:23.5英寸 1920*1080
    没有设置body的外边距内边距

    Window 对象表示浏览器中打开的窗口
    Screen 对象包含有关客户端显示屏幕的信息

    window.innerWidth 1920
    window.innerHeight 1019
    浏览器内容窗口的当前宽高
    宽度-包含竖滚动条
    高度-只有当前可见网页部分,不包含需要滚动才显示的部分

    document.documentElement.clientWidth 1905
    document.documentElement.clientHeight 1019
    浏览器内容窗口的当前宽高
    宽度-不包含竖滚动条
    高度-只有当前可见网页部分,不包含需要滚动才显示的部分

    document.body.clientWidth 1885
    document.body.clientHeight 8408
    body当前宽高
    宽度-内容+内边距
    高度-包含需要滚动才能显示的部分

    window.screenLeft:0
    window.screenTop:23
    整个浏览器相对于屏幕左上角的偏移

    screen.width 1920
    screen.height 1080
    显示屏的总宽高

    screen.availWidth 1920
    screen.availHeight 1053
    显示屏的宽高，以像素计，减去界面特性，比如窗口任务栏

json

回到目录 
    * JSON：JavaScript 对象表示法
    * JSON 是存储和交换文本信息的语法。类似 XML。
    * JSON 文本格式在语法上与创建 JavaScript 对象的代码相同。
    * JSON不是一种语言，它仅仅是一种数据的表达方式，从某种程度上来讲，JSON跟XML没有什么区别。
    * json文件类型 - xxx.json
    * JSON 文本的 MIME 类型是 "application/json"
    *
    * json - 一种语法,"key":"value" value可以是null
    * json对象 - {"key":"value",...} (包含多个键值对,与firstName = "John";lastName = "Doe";等价)
